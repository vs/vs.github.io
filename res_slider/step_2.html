Step two: Creating the mirror

Part A

Navigate to the unpacked subgit directory and run the next command in CMD/BASH with matching arguments to set the Git/SVN mirror up:

subgit configure --layout auto SVN_URL REPOS.GIT

Where:

  • SVN_URL is a link(url) leading to the SVN repository
  • REPOS.GIT is a path leading to dirrectory where the Git/SVN mirror will be created and configurated
